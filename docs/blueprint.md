# **App Name**: Health Passport

## Core Features:

- User Authentication: Secure user registration and login system with optional MFA.
- Document Upload: Patients can securely upload medical documents (PDF, JPG, PNG).
- AI-Powered Tagging: Tool that suggests relevant tags and categories for uploaded documents.
- Manual Data Entry: Structured forms for patients to manually input key medical information.
- Emergency QR Code: Generate a scannable QR code that displays designated emergency profile information, bypassing full login.
- Record Tagging: Categorize and add custom tags to records for easy organization.
- Secure Record Sharing: Secure URLs for sharing selected medical records with configurable expiration times and optional PIN protection.

## Style Guidelines:

- Primary color: Deep teal (#16A385), providing a sense of security and trustworthiness, and alluding to traditional healthcare aesthetics without being cliched.
- Background color: Light desaturated teal (#E0F2F1), creates a calm and professional backdrop.
- Accent color: Sky blue (#7DD3FC), to highlight interactive elements and important information.
- Body and headline font: 'Inter' sans-serif font, known for its clarity and modern, neutral appearance suitable for both headlines and body text.
- Simple, clear, and recognizable icons to aid navigation and highlight key features.
- Clean, intuitive layout with a focus on mobile-first responsiveness, ensuring accessibility across devices.
- Subtle transitions and animations to provide feedback and enhance user experience without being distracting.
